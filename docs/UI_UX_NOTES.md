# UI/UX Notes v10

## Arah Desain

Versi ini didesain sebagai Prompt Studio, bukan dashboard admin biasa.

Prinsip visual:

- Dark premium sebagai default.
- Glass panel secukupnya, tidak berlebihan.
- Sidebar sebagai project switcher.
- Panel tengah untuk brief builder.
- Panel kanan untuk output, score, dan engine breakdown.
- Mobile memakai bottom dock agar aksi utama mudah dijangkau.

## Struktur UX

1. User memilih mode project.
2. User mengisi brief berdasarkan kategori.
3. Prompt final muncul realtime.
4. Engine menilai kelengkapan brief.
5. User copy, save, download, atau export project.

## Elemen Kunci

- Prompt DNA: visualisasi fondasi prompt.
- Prompt Health: score kualitas brief.
- Engine Breakdown: menunjukkan bagian yang sudah terbaca.
- Kitab Prompt: penjelasan dasar istilah.
- Prompt Library: menyimpan versi lokal.
